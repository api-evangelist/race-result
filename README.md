# RACE RESULT (race-result)
The RACE RESULT REST interface allows retrieval of data from systems uploading to RACE RESULT servers, including RACE RESULT decoders that upload via TCP and Track Boxes, and supports remote control commands sent to decoders. Communication with the REST API requires API keys for the Customer ID being accessed.
